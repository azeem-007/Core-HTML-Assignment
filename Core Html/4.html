Ordered List of HTML Tags

  1. "a" Creates a hyperlink to another web page or resource
  2. "p" tag represents a paragraph of text
  3. "img" Embeds an image in the HTML document.
  4. "div" A container that is often used to group other HTML elements and apply styles.
  5. "h1-h6" Defines heading levels, with h1 as the highest and h6 as the lowest.
